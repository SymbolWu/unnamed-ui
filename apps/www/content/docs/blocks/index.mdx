## 使用说明

- **Blocks** 为原语层，主要提供样式和布局，不含业务逻辑
- **Composed** 为组合层，由 blocks 组合而成，包含完整交互逻辑
- 从左侧导航按分类浏览，或使用搜索快速定位组件

> **重要提示**：各组件文档中的示例代码路径（如 `@/components/...` 或 `@/registry/...`）需根据您项目的实际配置（如 `components.json`、路径别名等）进行调整，请勿直接复制使用。
