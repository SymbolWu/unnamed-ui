## 概述

Report Card 组件用于展示数据报告、指标卡片等信息，提供清晰的数据可视化界面。

## 快速开始

```tsx
import { ReportCardPrimitive } from "@/registry/wuhan/blocks/report-card/report-card-01";

export function Example() {
  return (
    <ReportCardPrimitive>
      <div>报告卡片内容</div>
    </ReportCardPrimitive>
  );
}
```
