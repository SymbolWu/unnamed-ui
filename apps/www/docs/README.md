## 💡 文档维护

### 更新文档
- 功能变更后及时更新对应文档
- 保持代码示例与实际代码同步
- 定期审查文档准确性

### 添加新文档
1. 确定文档类型，放入对应目录
2. 使用中文文件名，清晰描述内容
3. 更新对应目录的 README.md
4. 更新本文档的目录列表

### 文档规范
- 使用 Markdown 格式
- 组件文档使用 MDX 格式
- 保持统一的文档结构
- 添加清晰的标题和章节
