## 🚀 快速开始

1. **首次开发组件** → 阅读[组件开发指南.md](组件开发指南.md)了解架构
2. **设计原语组件** → 参考[原语设计指南.md](原语设计指南.md)
3. **创建新组件** → 使用[CLI工具](CLI工具使用指南.md)快速生成结构

## 💡 提示

- 原语 = 样式 + 结构，不包含业务逻辑
- Composed 组件封装业务逻辑，调用原语组件
- 使用 CLI 工具可以大大提高开发效率
